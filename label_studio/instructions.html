Instructions

NOTE: Label studio prevents links from automatically opening in a new tab/window, use right-click “Open in new tab” to open the linked PDFs if needed.

This is just a small overview of the labels, for more detailed instructions (including navigating the interface) you can download the annotation guidelines as PDF.

To help with character identification, see these lists with characters in the German movies and this list with characters in the Italian movies (data for both from IMDB).

One of the German movies, 8 Tage im August, contains occasional dialogue snippets in English. Those are not considered for this task (the models did not have access to the audio).

Additionally, the timing of the descriptions is not part of the evaluation, the models use the same timings as the original descriptions created by the audio describer.

Labels

Content:

irrelevant
AD contains unnecessary information
missing
AD misses crucial information or is too vague, e.g. she holds an object
redundant
AD contains redundant information, e.g. the phone is ringing, but the ringing is heard in the audio
subjective and/or patronizing
AD is not neutral, e.g. the kitchen is a mess instead of dirty dishes are stacked on the counter.
wrong action
AD has an action that is wrong, e.g. the man is walking but in the clip, the character is running
wrong object
AD has an object that was not correctly identified, e.g. the man is holding a phone but in the clip, the character is is holding a book
text missing
AD is missing relevant information from text shown in the scene, this could be text on a note, in a book or on a phone screen in the movie itself, or a text overlay (e.g., credits, or 5 years earlier to introduce a flashback scene).
other inacurracy
AD contains wrong information of another type, not covered by the given categories (not a wrong action, object or character)

Grammar:

ungrammatical/not fluent
AD has grammatical errors or awkward phrasing, e.g Gregi betrachtet den Kuchen, Wanda ihn.
too long/complex
The description is too long and/or too complex, e.g., with nested sentences.
wrong tense
AD is not in present tense
English wording/phrase (Denglisch/Itanglese)
AD contains phrases or words that are literal translations from English, e.g. Sie betrachtet sich im Spiegel und justiert ihr Haar, lei si rompe giù in lacrime.
other AD standards violation
The description violates AD standards or guidelines in some other form that is not covered by the other tags.

Coherence:

contextual gap
AD has an information gap to the previous AD, there is a logical jump that makes it confusing:
AD1: Paul and Lisa are talking to each other in front of the car.
AD2: Paul closes the door and leans back.
→ Paul getting in the car was not mentioned.
name repeated
AD repeats the name from the previous AD, example:
AD1: Paul sits down at the table
AD2: Paul starts eating
content repeated
AD repeats content from previous AD, example:
AD1: Paul is talking on the phone, walking down the street.
AD2: Paul talks on the phone.
makes no sense/unrelated content
The description makes no sense with respect to the clip and/or has completely unrelated content.
other incoherence
AD is not properly 'connected' to the previous AD, not none of the other labels apply.

Characters:

wrong character
AD has a character name that is not in the clip
wrong pronoun
AD has a wrong pronoun, e.g. she instead of he
redundant
AD has the correct character, but uses first and last name or some additional description that makes it sound unnatural
missing name
AD contains a description or a pronoun when the name was needed
bad description
AD describes a character by physical attributes, but is:
  • wrong (a blonde man, but the person has dark hair)
  • inappropriate
  • too vague to be useful
misattributed action
characters named in the AD are in the scene, but are not correctly associated with the action, example:
  • scene is A looks at B
  • AD is: B looks at A
  • scene is A and B are walking through a park, B is smoking
  • AD is: A and B are walking through a park, A is smoking
